AccountingBase (AccBase) - свободный проект по созданию учетной основы (платформы) по лицензии GPL.

Проект задуман как платформа, основа учета, максимально отвязанный от нюансов и деталей учетного и налогового законодательства любой конкретной страны (например, автор проекта находится в Республике Казахстан). Для этого создана уникальная структура прикладной учетной базы, не имеющая аналогов в других учетных программах. Учет понимается в самом широком смысле - как финансовый, управленческий, CRM, бюджетный и т.д. вид учета.

Предполагается, что развитие проекта может ветвиться, и идти в любой стране, в том числе странах бывшего Советского Союза. Благодаря свободной лицензии GPL любой сможет как создавать продукты на основе AccBase, так и использовать AccBase в учете любого предприятия.

Дальнейшее развитие проекта автором задумано в направлении MS SQL + .NET + C#. Однако, в случае каких-либо неблагоприятных для проекта поворотов и событий со стороны вендоров любого нижележащего базового ПО, проект задуман так, чтобы можно было переориентироваться на иные языки и иные DBMS.

Состав проекта - 
- AccBaseSpecification.xlsx - спецификация проекта, описание таблиц и полей. Приведены краткие пояснения логики проекта.
- AccBase.accdb - прототип проекта на MS Access (только таблицы). Начальные данные заложены для учета в Республике Казахстан.
- AccBaseSQL.zip - автономная база MS SQL Для подключения необходимо разрешить автономные базы на сервере SQL (свойства \ дополнительно \ разрешить автономные базы данных \ true).
- articles.zip - сборник статей автора, во многих из которых освещаются принципы проекта AccBase.
