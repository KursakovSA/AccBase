AccountingBase (AccBase) - свободный проект по созданию учетной основы (платформы) по лицензии GPL.

Проект задуман как платформа, основа, и костяк учета, максимально отвязанный от нюансов и деталей учетного и налогового законодательства любой конкретной страны (например, автор проекта находится в Республике Казахстан). Предполагается, что развитие проекта может ветвиться и идти в любой стране, в том числе странах бывшего Советского Союза.

Предполагается дальнейшее развитие проекта в направлении MS SQL + .NET + C#. Однако, в случае каких-либо неблагоприятных для проекта поворотов и событий со стороны вендоров любого базового ПО, проект задуман так, чтобы легко можно было переориентироваться на иные языки, иные DBMS, и иные базовые ПО более низкого уровня, без большого ущерба и затрат.

Составные файлы - 

- AccBaseSpecification.xlsx - спецификация проекта. Начальные данные заложены для учета в Республике Казахстан.

- AccBase.accdb - прототип проекта.

- AccBase1.mdb, AccBase1.ldb - база SQL с начальными данными (ожидается).
